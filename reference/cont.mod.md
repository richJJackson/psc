# Example model for a survival outcome

A generated model with a continuous data endpoint and a identity link
function. Data for the model were synthetically generated and are based
on a dataset to evaulate the use of Sorafenib in HCC akin to the PROSASH
model (see ?psc::surv.mod for more details)

## Usage

``` r
cont.mod
```

## Format

A model of class 'glm':

- ecog:

  ECOG performance Status

- logafp:

  AFP - log scale

- alb:

  albumin

- logcreat:

  Creatinine - log scale
