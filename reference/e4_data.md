# Example Dataset of patients treated with GemCap in the ESPAC-4 trial

A dataset containing 346 simulated patients. Data are based on the
patietns randomised to revceive GemCap in the ESPAC-4 trial

## Usage

``` r
e4_data
```

## Format

A model of class 'flezsurvreg':

- time:

  survival time

- cen:

  censoring indicator

- nodes:

  negative (n=1) or positive (n=2) lymph nodes

- grade:

  tumour grade (1,2 or 3)

- lca199:

  log transformed ca19.9

- t:

  T-stage (1,2 or 3)
