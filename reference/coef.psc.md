# Returns the coefficient estimate of a psc object.

Returns basic measures of the posterior distribution obtained from the
psc object

## Usage

``` r
# S3 method for class 'psc'
coef(object, ...)
```

## Arguments

- object:

  a 'psc' object

- ...:

  not used

## Value

The summary of the posterior distribution for the efficacy parameter in
terms of the median and 95
